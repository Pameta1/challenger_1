# Análisis de Ventas, rendimiento y reseñas de 4 tiendas de Alura Store.

Creación de Proyecto con el objetivo de que el propietario de las Tiendas Alura Store don Juan, pueda vender e invertir en un nuevo negocio.
Se evaluará el desempeño menor de cada una de las cuatro tiendas que posee don Juan.
